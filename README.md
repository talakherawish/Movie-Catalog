# Movie Catalog

A simple Java-based desktop application that allows users to browse, manage, and explore a catalog of movies. Built with modular design and file-based data storage, this project is ideal for learning object-oriented programming and basic GUI development.

## Features

- Movie listing: Displays a list of movies from a text file.
- Modular structure: Organized into packages like `application` and `images`.
- File-based storage: Reads movie data from `movies.txt`.
- Image support: Includes an `images` folder for movie posters or icons.
- JavaFX-ready: Compatible with JavaFX for future GUI enhancements.

## Project Structure

src/ 
├── application/ # Core Java classes and logic 
├── images/ # Movie-related images 
├── movies.txt # Movie data file 
├── module-info.java # Java module descriptor 
├── .classpath/.project # Eclipse project files
